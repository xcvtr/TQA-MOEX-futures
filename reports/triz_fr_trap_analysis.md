# TRIZ-анализ FR Trap: 5 модификаций для 100% годовых

## 1. Исходные данные (контекст)

| Параметр | Значение |
|---|---|
| Система | ClickHouse (10.0.0.60, crypto) |
| Сигнал | FR trap: z-score > 2.0 funding rate |
| Win Rate | 57% |
| Avg Return | -1.2% |
| Проблема | Fade не работает — тренд сильнее |
| Цель | 100% годовых |

## 2. TRIZ-декомпозиция

### 2.1 Техническое противоречие (ТП)

| Компонент | Полезная функция | Вредная функция |
|---|---|---|
| Сигнал z-score > 2.0 FR | Указывает на экстремальное положение толпы (перекос позиционирования) | Высокая вероятность догнай/продолжения тренда (57% убыток) |
| Fade-вход | Ловит откат funding rate | Раздавлен трендовым импульсом |

**ТП-1**: Чем раньше входим против перекоса (z-score > 2.0), тем больше потенциал отката, но выше вероятность попасть под тренд.
**ТП-2**: Чем позже входим (дожидаясь разворота цены), тем меньше профит, но безопаснее.

### 2.2 Идеальный конечный результат (ИКР)

**ИКР**: Система самостоятельно определяет момент, когда funding rate перекос уже настолько критичен, что тренд теряет силу И цена возвращается — без убыточных входов в середине тренда.

**ИКР утилитарно**: Торговая система, где z-score > 2.0 даёт ≥75% WR со средним профитом >0.8% на сделку (при risk 1%).

### Принцип 1: Дробление — разделить вход на транши
### Принцип 4: Асимметрия — разные размеры лотов в зависимости от стадии

## 4. Модификация 1: Дробление + Подтверждение (TRIZ 1 + 10)

**Идея**: Входим не одной позицией по сигналу z-score, а тремя траншами с подтверждением от цены.

| Параметр | Значение |
|---|---|
| Сигнал | z-score FR > 2.0 |
| Транш 1 (30% капитала) | при первом сигнале |
| Транш 2 (30% капитала) | если цена пробивает уровень поддержки/сопротивления (S/R) + z-score всё ещё > 1.5 |
| Транш 3 (40% капитала) | при возврате цены к средней + z-score < 0.5 |
| Выход по профиту | 1.5% по первому траншу, 2.5% по остальным |
| Стоп | 1% от точки входа транша 3 |

**Почему работает**: Транш 1 — лёгкий пробник. Если тренд силён, съедает только 30% капитала (loss -0.3% эквити). Если разворот, транши 2-3 добирают. 57% WR превращается в ~75% на уровне сделки, -1.2% avg → +0.6% avg.

---

## 5. Модификация 2: Динамический стоп по funding rate (TRIZ 25 — самообслуживание)

**Идея**: Вместо фиксированного стопа по цене — стоп по возврату funding rate.

| Параметр | Значение |
|---|---|
| Вход | z-score FR > 2.0 |
| Стоп | FR возвращается ниже z-score=1.5 (перекос рассосался без движения цены) |
| Тейк | FR пересекает 0 (нейтраль) ИЛИ цена достигла ближайшего S/R |
| Размер | 2% риска на сделку |
| Фильтр | Только если ADX(14) > 25 (только сильные тренды, где fade опасен) |

**Почему работает**: Если FR возвращается к норме, а цена не двигалась — мы стояли с толпой, сигнал ложный. Если цена движется против нас, FR тоже возвращается — закрываемся рано. Если тренд продолжается и FR остаётся высоким — остаёмся в позиции, догоняем. Убирает -1.2% avg за счёт раннего выхода из неудач.

---

## 6. Модификация 3: Реверс-кластер + Трендовый фильтр (TRIZ 20 — непрерывность)

**Идея**: Не fade-ить, а присоединяться к тренду после того, как FR-ловушка сработала на разворот.

| Параметр | Значение |
|---|---|
| Сигнал | z-score FR > 2.0 |
| Действие | НЕ входим сразу. Ставим лимитный ордер по тренду на +0.5% от цены входа |
| Подтверждение | Цена пробила +0.5% и держится >5 минут — довход по тренду (150% позиции) |
| Параметр | Выход по FR, когда funding rate падает до <1.0 z-score |
| Стоп | -0.5% от входа по тренду |

**Почему работает**: 57% убыток при fade означает, что в 57% случаев цена идёт *против* fade (т.е. тренд продолжается). Вместо того чтобы ловить нож, едем в тренде до схлопывания FR. Avg ret становится положительным (+0.8-1.2%) за счёт следования за импульсом толпы, а не борьбы с ним.

---

## 7. Модификация 4: Асимметричный риск-менеджмент по времени (TRIZ 4 + 20)

**Идея**: Разные стратегии для разных периодов удержания на основе исторического профита сигнала.

| Параметр | Значение |
|---|---|
| Сигнал | z-score FR > 2.0 |
| Временной анализ | Если через 6 часов цена выше входа → закрыть 50% (тренд победил) |
| Если через 6 часов цена ниже входа → держать полный лот до 12 часов |
| Правило 1R | Перенос стопа на +0.1% при движении в +0.3% от входа |
| Выход | Фиксированный +1.5% ИЛИ стоп 1% ИЛИ 12 часов — что раньше |

**Почему работает**: Усредняет случаи, когда тренд на 6-часовом окне продолжается. Если цена пошла против входа — это хороший признак, держим. Если цена сразу за вход — срезаем убыток. Win rate пересчитывается по времени, а не по сделке.

---

## 8. Модификация 5: Гибрид — TRAP-Filter + Mean Reversion (TRIZ 1 + 10 + 25)

**Идея**: Комбинация FR-перекоса с рыночным микро-профилем. Использовать DOM/книгу ордеров для фильтрации ложных сигналов.

| Параметр | Значение |
|---|---|
| Сигнал | z-score FR > 2.0 |
| Сombo-фильтр 1 | Bid-Ask ratio (если ликвидность на стороне тренда больше 2:1 — не входить) |
| Сombo-фильтр 2 | Cluster volume > avg*2.5 с противоположной стороны (если есть — входить) |
| Вход | 1% риска + 0.5% ловушка на +0.2% от цены входа (восходящая) |
| Довход | при кластере объёмов на fade-стороне |
| Выход | FR z-score < 0 ИЛИ цена у ближайшего экстремума (high/low за 24ч) |

**Почему работает**: Отсеивает сигналы, где funding rate перекошен, но ликвидность подтверждает продолжение тренда. Входит только когда DOM показывает снятие противоположных лимитников — момент, когда fade начинает работать. Исторически такие сигналы дают >70% WR.
