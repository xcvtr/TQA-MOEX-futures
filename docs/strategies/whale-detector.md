## Идея

Розничные инвесторы (FIZ) в среднем покупают, институционалы (YUR) продают.
Когда FIZ начинает массово закрывать позиции или убегать — это сигнал,
что розница паникует и рынок скоро развернётся.

### Паттерны (бычьи — ожидаем рост)

| Паттерн | Условие |
|---------|---------|
| FIZ_DROP_3D | Количество FIZ счетов падает 3+ дня подряд |
| YUR_LOAD_5D | Средний размер позиции YUR растёт 5+ дней |
| FIZ_FLEE_3D | FIZ счета упали на >5% за 3 дня |
| FIZ_DROP_EXTREME | z-score < -2 |
| FIZ_SHORT_SURGE | FIZ шорты выросли >5% за 3 дня |

### Паттерны (медвежьи)

| Паттерн | Условие |
|---------|---------|
| FIZ_EUPHORIA | Доля FIZ в лонгах растёт 5+ дней |
| FALLING_KNIFE | Цена -1% + FIZ счета >2% за 5d |
| RALLY_FLEE | Цена +1% + FIZ счета <-3% за 5d |
| FIZ_OVERLOAD | FIZ на 95+ перцентиле + цена растёт |
| FIZ_PEAK | z-score > +2 |
