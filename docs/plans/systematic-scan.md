# План: Системный поиск работающих стратегий на ВСЕХ 47 тикерах MOEX

## Проблема
BR SMA mean reversion даёт Calmar 5.08 без комиссий, но комиссия 2 RUB/контракт убивает (5× edge на контракт).
RI работает (Calmar 0.72 с комиссией) из-за высокого GO. 
Нужно проверить ВСЕ 47 тикеров системно.

## Что сделать

### Шаг 1: Данные по всем 47 тикерам
Для КАЖДОГО из 47 тикеров (список из TICKER_CONFIGS):
- Загрузить 5m OHLCV из БД (host=10.0.0.64 db=moex) через load_ohlcv(ticker, 365*3) или прямой SQL
- Resample до daily (D)
- Для тикеров без 5m данных: попробовать daily таблицу, пропустить если нет

### Шаг 2: Для КАЖДОГО тикера — тест SMA mean reversion
Сигнал: SMA5(close) < SMA20(close) → LONG
Hold: 5 дней
Stop-loss: 10%
Margin usage mu: 0.50
Комиссия: 2 RUB/контракт

Через DailyPortfolio (scripts/daily_bar_level.py):
- return%, DD%, Calmar
- Сколько сделок
- Сколько контрактов в среднем на сделку
- Комиссия % от капитала

### Шаг 3: Если Calmar > 0 — walk-forward
4 folds, проверить стабильность

### Шаг 4: Если Calmar > 0 — sweeп параметров
mu=[0.10, 0.20, 0.35, 0.50], hold=[5, 10, 21, 30], sl=[0.05, 0.10]

### Шаг 5: Отчёт
Единая таблица: ticker | GO | return% | DD% | Calmar | trades | avg_contracts | comm_pct | WF

## Требования
- Полностью автоматически, без остановок на вопросы
- Каждый тикер: своя строка в таблице
- Если данные недоступны: "NO DATA"
- Если сделок < 5: "NO SIGNALS"
- Если Calmar < 0: красным
- Если Calmar > 0: зелёным
- Время: ~2-3 часа на все 47 тикеров
- Отчёт: reports/2026-06-11-systematic-scan.md
