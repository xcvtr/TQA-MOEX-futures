## Ключевые выводы

1. OI имеет 5-минутное разрешение для MX, SR, GD, RI, LK (~472k записей). Si - исключение (daily only).
2. H4 даёт в 20x больше сигналов, чем D1 (428 vs 22).
3. Общая WR H4 выше (59% vs 55%).
4. Для RI и LK D1 бесполезен - 0% WR. H4 даёт 57% WR на 7 и 60 сигналах.
5. 100% LONG - ни одного SHORT сигнала. BEARISH паттерны не добирают dominance 3x.

## Проблемные зоны

- GD на H4: 343 сигнала с 59% WR, но средняя доходность только +0.18% на сделку. D1 даёт 71% с +1.10%. Для GD лучше D1.
- Отсутствие SHORT: стратегия односторонняя. Нужна калибровка BEARISH паттернов (понизить dominance до 2x).
- Пороги не универсальны: каждый тикер требует индивидуального scale.
