# Техническое задание: Визуализатор манипуляций MOEX с разделением OI физ/юр

## Цель
Создать скрипт, который:
1. Загружает 5m свечи + OI (физлица/юрлица) для MOEX фьючерса
2. Находит манипулятивные паттерны (дивергенция OI потоков, OI-ловушки, экстремумы)
3. Строит график: цена + OI с разделением физ/юр + входы/выходы

## График (matplotlib)

### Панель 1: Цена + паттерны
- Свечной график OHLC (mplfinance) или линия close
- Свинговые уровни (хаи/лои пунктиром)
- Маркеры паттернов:
  - FLOW_DIVERGENCE — ромб 🔶 (физ/юр дивергенция)
  - OI_TRAP — треугольник △
  - OI_EXTREME — звезда ✦
  - FALSE_BREAK — круг ○
  - STOP_HUNT — крест ✕
  - VOL_CLIMAX — квадрат □
- BULL (зелёный) / BEAR (красный)
- Вход → выход: соединительная линия от входа к выходу

### Панель 2: OI физлица (net + buy/sell)
- Линия fiz_net (net = buy - sell)
- buy/sell отдельными линиями (опционально)
- Цвет: синий для физлиц

### Панель 3: OI юрлица (net + buy/sell)
- Линия yur_net
- buy/sell отдельными линиями
- Цвет: оранжевый для юрлиц

### Панель 4: Flow z-score
- fiz_flow_zscore (экстремальный поток физлиц)
- yur_flow_zscore
- Пороги ±2 штриховой линией

## Детекторы

### 1. FLOW_DIVERGENCE (новый)
Когда физлица резко наращивают buy, а юрлица sell (или наоборот):
- buy_FIZ_flow_zscore > +2 И sell_YUR_flow_zscore > +2 → дивергенция
- Или fiz_flow > 0 И yur_flow < 0 с z-score > 1.5

### 2. OI_TRAP (существующий)
Цена и fiz_net в одном направлении, затем разворот.

### 3. OI_EXTREME (существующий)
|fiz_zscore| > порог + разворот цены.

### 4. FLOW_EXTREME (существующий)
|fiz_flow_zscore| > порог + разворот.

## Входы/выходы
Для каждого паттерна:
- **Вход**: время паттерна, цена close
- **Выход**: 6 часов спустя (72 свечи), цена close
- Соединительная прямая линия на графике

## Параметры CLI
```
--symbol Si        # тикер
--days 60          # глубина
--zscore 2.0       # порог
--output chart.png # файл
--no-show          # не открывать окно
```

## Формат вывода
- PNG файл
- В консоль: таблица найденных паттернов с метриками успеха
