# Формат CHANGELOG

Изменения фиксировать в корневом `CHANGELOG.md`. В нём — одна строка-ссылка на детальную запись:

```markdown
# CHANGELOG

Текущая запись: [2026-06-06 — Описание](docs/changelog/2026-06-06-описание.md)
```

Детальные записи — в `docs/changelog/` с датой в начале имени файла:
```
docs/changelog/
├── 2026-06-06-securities-and-oi-fix.md
└── ...
```

Формат детальной записи: заголовок с датой, секции «Сделано», «Файлы», «Что не сделано».
