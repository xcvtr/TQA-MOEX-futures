## Суть стратегии: Volume Surge + FIZ/YUR Divergence

Розница (FIZ) и киты (YUR) торгуют в противоположные стороны. Когда происходит вспышка объёма на фоне расхождения потоков — сигнал, что одна сторона права, а другая паникует.

### Сигнал LONG (fiz_short_yur_long):
1. FIZ продаёт (fiz_z < 0) — розница паникует
2. YUR покупает (yur_z > 0) — киты набирают позицию
3. Volume Surge (vol_z ≥ 2.5-3.0)
4. Вход: LONG на open следующего 5m бара
5. Выход: close через N баров (60m-240m)
6. Win = return > 0

### Сигнал SHORT (fiz_long_yur_short):
1. FIZ покупает (fiz_z > 0) — розница лезет в лонг
2. YUR продаёт (yur_z < 0) — киты распределяют
3. Аналогично
