## Суть стратегии: Volume Surge + FIZ/YUR Divergence

**Базовый принцип:** Розница (FIZ) и киты (YUR) торгуют в противоположные стороны. Когда происходит вспышка объёма на фоне расхождения потоков — это сигнал, что одна сторона права, а другая паникует.

**Логика сигнала LONG (fiz_short_yur_long):**
1. FIZ продаёт (fiz_z отрицательный, FIZ в шорте) — розница считает, что рынок упадёт
2. YUR покупает (yur_z положительный, YUR в лонге) — киты набирают позицию
3. **Volume Surge** (vol_z ≥ порога) — вспышка объёма подтверждает, что событие значимое
4. **Divergence** (fiz_z и yur_z разных знаков, |div_z| ≥ порога) — потоки расходятся
5. Вход: **LONG** на открытии следующего 5m бара
6. Выход: через N баров (оптимальный горизонт)
7. Win = return > 0 (close-based, никаких трейлинг-стопов)

**Логика сигнала SHORT (fiz_long_yur_short):**
1. FIZ покупает (fiz_z положительный) — розница лезет в лонг
2. YUR продаёт (yur_z отрицательный) — киты распределяют позицию
3. Вход: **SHORT**
4. Всё остальное аналогично
