# Checkpoint 041 — Dashboard localStorage Persistence

**Дата:** 2026-06-12

## Что сделано

1. **Обнаружена проблема:** При обновлении дашборда (F5) сбрасывались:
   - Выбранный тикер
   - Таймфрейм
   - Количество дней
   - Чекбокс «Показать сделки»
   - Активная вкладка

2. **Решение:** Добавлено сохранение состояния в `localStorage` и восстановление при загрузке страницы.

3. **Изменённые контролы:**
   - `<select id="live-ticker">` — тикер
   - `<select id="live-tf">` — таймфрейм (5m, 15m, H1, H4)
   - `<input id="live-days">` — количество дней
   - Не затронуто: вкладка Backtest (там свои контролы)

4. **Реализация:**
   - Функция `saveState()` — сохраняет ticker, tf, days в `localStorage`
   - Функция `restoreState()` — восстанавливает при `DOMContentLoaded`
   - Привязка `onchange` на все три контрола для автосохранения
   - `loadLive()` теперь с задержкой `setTimeout` после восстановления (чтобы DOM успел инициализироваться)
