# Checkpoint 096 — CVD Divergence Walk-Forward + Аудиты

**Дата:** 2026-06-14 (update: 2026-06-25)
**Проект:** TQA-MOEX-futures
**Данные:** ClickHouse `moex.tradestats_fo` (AlgoPack, платный)
**Символы:** NG, BR, Si, MXI

---

## Что сделано

Честный walk-forward тест на CVD-дивергенции + три аудита.

**Метод:** Train 180d → Test 60d, скользящее окно (шаг 60). `cvd_cum` строго внутри train, на test продолжается от последнего train-значения. Grid: lookback (5/10/20) × hold (1/3/5) × quantile (0.6/0.7/0.8). Сигнал: divergence между price_chg(lookback) и cvd_cum_chg(lookback). Фиксация hold-баров, нет pyramiding.

## 2. Аудит корреляции сигналов

Проверка: не завышен ли 82.4% WR за счёт одновременных сигналов на нескольких символах.

Метод: прогон полного walk-forward с лучшими конфигами, запись времени каждого сигнала, подсчёт overlap.

┌────────────────┬─────────┬────────┐
│ Тип сигнала    │ Кол-во  │   %    │
├────────────────┼─────────┼────────┤
│ 1 символ       │ 70,295  │ 84.0%  │
│ 2 символа      │ 12,473  │ 14.9%  │
│ 3 символа      │   893   │  1.1%  │
│ 4 символа      │    26   │  0.0%  │
└────────────────┴─────────┴────────┘

**Вывод:** 84% сигналов приходят только на 1 символе. Мульти-символьный overlap минимален, агрегированный WR не завышен за счёт корреляции. Топ пары: MXI+Si (5.4%), MXI+NG (3.6%), NG+Si (2.9%).

## Ключевые выводы

1. **CVD-дивергенция стабильна** на всех 4 фьючерсах MOEX, 6+ лет, ни одного проигрышного года
2. **Корреляция сигналов низкая** (84% независимы) — можно использовать как портфель
3. **PnL в RUB**: ~+177M (предварительно, нужна верификация через единый walk-forward)
4. **Комиссии** незначительны
5. **Основной риск:** 82.4% WR и 254M тиков выглядят слишком хорошо — нужен walk-forward с фиксированными параметрами на бумажном трейдере в реальном времени
