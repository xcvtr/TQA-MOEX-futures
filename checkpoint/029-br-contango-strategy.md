## Approach A: Futures Basis

**Сигнал:** basis = (close_front - close_back) / close_front < -threshold → LONG

Проблема: базис доступен только на 100 датах с оверлепом. Forward-fill даёт устаревший сигнал для остальных 839 дней.

## Approach B: SMA Proxy (Mean Reversion)

**Сигнал:** basis = (SMA5 - SMA20) / SMA20 < -threshold → LONG

Суть: покупка при SMA5 < SMA20 (цена ниже среднего за 20 дней), держать N дней.

**6/27 комбинаций стабильны во всех 4 folds.** Все — с hold=5 и sl≥0.05.

### Почему это работает

1. **SMA5 < SMA20** = цена опустилась ниже 20-дневного среднего = перепроданность
2. **Hold=5** = захват отскока (mean reversion) без передержки
3. **Stop-loss 10%** = защита от чёрных лебедей (использовано 2 раза из 94)
4. **mu=0.50** = 50% капитала в позиции (консервативно для 10% стопа)
