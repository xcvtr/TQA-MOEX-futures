# Чекпойнт 088: CVD Divergence — реалистичная модель исполнения

## Что изменилось

### 1. Исправлены спецификации MOEX (Critical)

└─────────┴────────────┴────────────┴────────────┘

### 2. Реалистичная модель исполнения

- **Лимитный вход** с проверкой касания (touch-check через high/low бара)
- **Адаптивный сдвиг лимитки**: 30% от ATR(14), мин 5 тиков, макс 20
- **Выход** по close следующего 5м бара (hold=1)
- **Slippage**: 0.5 тика на вход + 1.0 тик на выход = 1.5 тика round-trip

### 3. Мульти-потоковая дедупликация (Critical)

Si (USD/RUB) торгуется в режимах TOD/TOM с разными ценами. Добавлена `deduplicate_1m()` — для каждой минуты выбирается запись с максимальным объёмом.

**Результат:** Аномалия Si exit=91515 устранена. Сделка #16: было +9,165₽ → стало -1,365₽ (реальный убыток).

### 4. Catchup защищён от look-ahead

`catchup_missed_signals()` теперь вставляет сигналы **только за текущий день**.

### 5. Создана библиотека `lib_cvd_divergence.py`

Единый источник правды для paper trader, бэктеста и restore-скриптов:
- TICK, TICK_COST, GO, SYMBOLS
- `resample_to_5m()`, `deduplicate_1m()`, `detect_signals()`
- `calc_entry_price()`, `check_touch()`, `calc_pnl_rub()`, `calc_slippage_ticks()`
- `simulate_trade()`, `walk_forward_split()`

### 6. Бэктест v4 обновлён

- Использует реальные pr_high/pr_low из ClickHouse (84% баров имеют ненулевой диапазон)
- Полный прогон: 33,631 сделок, WR 74.1%, Net PnL +28.5M RUB
- 70/70 месяцев положительны, Long/Short симметрия

### 7. Полный аудит

Результат: **9 сделок исполнились** (7 no-touch после дедупликации), **Capital: 95,623.93 ₽** (-4.4%) за 2026-06-26.
