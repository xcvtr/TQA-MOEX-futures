# 031 — Grid Search BASE Parameters: Score×Bars×Stop×Leverage (2026-06-15)

## Контекст

## Методология

- **Тикер**: GL (выбран как лучший, с самой длинной историей)
- **Период**: 2025-01-01 — 2026-05-01 (16 месяцев)
- **Данные**: moex.prices_5m + moex.prices_5m_oi + moex.openinterest (accounts)
- **Система**: bar-level simulation через один проход по барам (векторизованный grid) — все комбинации за 1 проход
