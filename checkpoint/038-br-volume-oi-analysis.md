# 038 — BR Volume + OI Time Analysis (M5)

Дата: 2026-06-12

## Что сделано
1. **Скрипт `scripts/br_volume_time_analysis.py`** — временной анализ объёмов:
   - Загрузка 138К M5 баров BR из ClickHouse
   - Volume z-score (20-bar окно)
   - Распределение всплесков по часам/дням/сессиям
   - Кластеризация всплесков
   - OpenCode сам исправил баг с numpy.timedelta64

2. **Скрипт `scripts/br_volume_oi_analysis.py`** — анализ Volume × OI вместе:
   - Загрузка 99К баров с OI (JOIN prices_5m × prices_5m_oi)
   - Классификация спайков по составу OI:
     - Yur Accumulation (19%) — юрлица набирают
     - Fiz Panic (19%) — физлица давят
     - Mixed (62%)
   - Физики long на 76% spike-баров
   - Price impact по каждому типу (WR ≤48%)

## Результаты
- Объёмные всплески на BR — **следствие**, а не причина движения
- Ни один тип спайка не даёт WR > 48% (хуже random)
- Структура OI (fiz_share) на спайках не отличается от среднего (~72%)
- Yur accumulation незначительно лучше fiz panic по avg return
- Дальше: смотреть контекст — OI до спайка, цена на экстремуме
