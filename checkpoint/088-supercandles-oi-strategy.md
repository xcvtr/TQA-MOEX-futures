# 088 — Supercandles × OI × FUTOI: корреляционный анализ + стратегия

**Дата:** 2026-06-22

## Контекст

Продолжение чекпойнта 087. После загрузки FUTOI/HI2/Alerts (087) решено добавить **Supercandles** — агрегированные 5m бары из AlgoPack fo (`tradestats_fo`). Обнаружены и исправлены проблемы с OI (был всегда 0) и дубликатами (разные контракты на 1 ticker).

## Что сделано

### 1. Исправлена таблица `moex.supercandles_fo`

**Проблемы:**
- `oi_change` всегда 0 (считался как `max(oi_change)` за день, а OI на 5m баре почти не меняется)
- Дубликаты: разные экспирации (BRM6, BRN6, BRQ6...) маппятся на один ticker BR, цены в разных масштабах

**Исправления:**
- Дневной OI считается как `oi_close_last - oi_open_first` через CTE `daily_oi`
- Для каждого `(ticker, date)` выбирается 1 **активный контракт** (максимальный дневной объём)
- Таблица: `moex.supercandles_fo` (9.16M строк, 66 тикеров, 2020-01 → 2026-06)

### 2. Корреляционный анализ Supercandles → ret_next

Для каждого тикера посчитаны lagged корреляции фич supercandles с доходностью следующего дня.

**Лучшие предикторы:**

| Фича | Лучшие тикеры | Corr |
|------|---------------|------|
| **OI_change → next** | **Si**: −0.17, **CR**: −0.14 | OI растёт → цена падает |
| **ret_5d → next** (mean rev) | Si: −0.10, CR: −0.10, GL: −0.09 | После 5d роста → падение |
| **disb_avg → next** | Si: +0.08, SR: −0.06 | Слабо |
| **net_vol_pct, vol_b_ratio, net_vol** | ≈ 0 | Не работают |

### 3. Стратегия OI + Mean Reversion

Сигналы:
- **OI z-score**: OI < −1.3σ → LONG next day; OI > +1.3σ → SHORT
- **MR z-score**: ret_5d < −1.3σ → LONG; ret_5d > +1.3σ → SHORT
- **COMBINED**: совпадение обоих сигналов

**Результаты (2020-2026, реинвест):**

| Сигнал | Всего | Ret | WR | DD |
|--------|-------|-----|----|-----|
| **OI_only** | 1077 сделок | **+400.7%** | 55% | −52.6% |
| MR_only | 1385 | −83.0% | 53% | −93.2% |
| ALL | 2655 | −13.6% | 54% | −88.5% |

**Проблема:** Стратегия нестабильна по периодам:
- OOS 2020-2024: −83.8%, DD −93.1%
- IS 2025-2026: **+434.3%**, DD −17.8%

**Лучший одиночный тикер — CR (CNY/RUB):**
- OI_only: +69.4% за весь период, стабильно по годам (кроме 2024)
- WR 57% LONG, 54% SHORT
- **Единственный тикер с положительной доходностью в OOS**
