# Checkpoint 094: BR 3-red exhaustion + vol climax

После системного отрицательного результата всех TRIZ-направлений найден **первый рабочий directional сигнал на BR**.

## Стратегия

**Сигнал:** 3 красные свечи подряд + vol_z > 2.5 (аномальный объём) → истощение → LONG
**Вход:** по open следующего бара
**Выход:** Take Profit = entry + ATR×0.75 ИЛИ Stop = entry − ATR×3.0 ИЛИ Max Hold = 120 баров

**Параметры:**
| Параметр | Значение |
|----------|----------|
| Таймфрейм | 5-min |
| Порог vol_z | > 2.5 |
| Цель | entry + ATR×0.75 |
| Стоп | entry − ATR×3.0 |
| Макс удержание | 120 баров (10 часов) |
| ATR период | 14 |

## Контекст поиска
