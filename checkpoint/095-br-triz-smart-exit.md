# Checkpoint 095: BR 3-red exhaustion + TRIZ smart exit

## Что сделано

К 3-red exhaustion сигналу (чекпойнт 094) добавлен **TRIZ-умный выход**:
- **Динамичность** — выход при спаде объёма (vol_z < 1.0) — MM перестали двигать
- **Проскок** — выход через 12 баров если цена не закрепилась выше entry
- **Матрёшка** — sma20 crossover как дополнительный сигнал выхода

Это вытянуло стратегию из отрицательного PnL при WR 65-73% в стабильный положительный.

## Стратегия

**Таймфрейм:** 15m (resample из 5-min supercandles_fo)
**Сигнал:** 3 красные свечи подряд + vol_z > порога → истощение → LONG
**Вход:** лимитка на уровне min за последние 4 бара (min4). При fill → open следующего бара.
**Выход:** комбинированный:
1. Take Profit = entry + ATR × target_mult
2. Stop Loss = entry − ATR × sl_mult
3. Vol decay: если vol_z падает < 1.0 после входа — выход (динамичность)
4. Proskok: если через 12 баров (3ч) цена ниже entry — выход
5. SMA cross: после того как цена была выше SMA20, пробила его вниз — выход

**Лучшие параметры:**
| Параметр | Значение |
|----------|----------|
| zv_th | 3.0 |
| target_mult | 2.0 |
| sl_mult | 1.5 |
| min_lookback | 4 |
| max_hold | 32 бара (8ч) |

## Контекст

Предыдущие попытки:
- **5m min5 entry** (чекп 094): WR 68%, но PnL +15K — не прошёл walk-forward
