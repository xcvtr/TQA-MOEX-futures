- **Order Block:** SBERF, BR (Сбер + Brent)

- ADX-фильтр в `cron_scanner.py` теперь загружает данные ОДИН раз на тикер (через `adx_data_cache`)

### 5. 📊 Rolling WR на дашборде
- Добавлен `_rolling_winrate()` — скользящая WR за последние 50 сделок
- На дашборде: sparkline-график + таблица последних 10 значений
- ⚠️ Предупреждение, если WR упала ниже 40%

### 6. 📦 Оптимизация equity dashboard (5057 порт)

- HTML должен уменьшиться с ~12MB до ~1-2MB
