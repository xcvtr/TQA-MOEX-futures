# 005 — Mean Reversion After Volatility Exhaustion (MRAVE)

**Дата:** 2026-06-08
**Статус:** Новая стратегия. Полный скрининг 59 тикеров MOEX за 6 месяцев.

---

## Полная логика входа и выхода

### 🎯 Идея

После **3+ последовательных баров** в одном направлении (импульс) цена часто **выдыхается**. Маркер исчерпания импульса:
1. Свеча с **аномально широким диапазоном** (range ≥ 1.5× медианы последних 50)
2. **Вспышка объёма** (vol_z-score ≥ 1.5)
3. **Закрытие в середине** диапазона (close position 30-70% от high-low)
4. **Предшествовало 3 бара чистого направления**

→ Заходим **против** направления этих 3 баров (fade the exhaustion).

### 📥 Вход

**Condition A — SHORT entry** (fade 3-bar up-move):
```
prev_changes = close[i-3:i] - open[i-3:i]
1. All 3 prev_changes > 0        → 3 бара роста
2. vol_z[i] ≥ 1.5                → объёмный всплеск
3. range[i] ≥ median(range)×1.5  → широкая свеча
4. 0.3 ≤ pos_in_range[i] ≤ 0.7  → close не в экстремуме
→ ENTRY SHORT @ close[i]
→ EXIT @ close[i+12] (≈60m), альтернативы: h=6 (30m) или h=24 (120m)
```

**Condition B — LONG entry** (fade 3-bar down-move):
```
prev_changes = close[i-3:i] - open[i-3:i]
1. All 3 prev_changes < 0        → 3 бара падения
2. vol_z[i] ≥ 1.5                → объёмный всплеск
3. range[i] ≥ median(range)×1.5  → широкая свеча
4. 0.3 ≤ pos_in_range[i] ≤ 0.7  → close не в экстремуме
→ ENTRY LONG @ close[i]
→ EXIT @ close[i+12] (≈60m)
```

### 📤 Выход

- **Фиксированный горизонт:** 6, 12 или 24 бара (5m = 30, 60, 120 мин)
- **Оптимальный:** 12 баров (60 мин) — баланс скорость/качество
- **Stop-loss:** не используется в стратегии (горизонт фиксирован)
- При горизонте 24 — выходим через 2 часа

### 📊 Ключевые метрики

```
pos_in_range = (close - low) / (high - low)  # 0.0 = close=low, 1.0 = close=high
vol_z = rolling_z_score(volume, window=20)   # z-score последних 20 баров
range = high - low
```

### ⚡ Отличие от Volume Surge

| Характеристика | Volume Surge | Mean Reversion |
|:---------------|:------------:|:--------------:|
| Торгуем | **с** импульсом | **против** импульса |
| Данные | fiz_buy/fiz_sell/yur_buy/yur_sell | только цена + объём |
| Нужен | MOEX OI table (moex_prices_5m_oi) | только moex_prices_5m |
| Работает на | тонких тикерах (OI < 230K) | **всех** тикерах |
| Сигналов/год | ~100-300 | ~30-200 |
| WR (passing) | 55-61% | **60-87%** |
| Лучшие | HS, KC, DX | NM, TN, TT, BR, SBERF |

### Ключевые находки

1. **TN и TT — идентичны.** TN = TT = фьючерс на древесину/пиломатериалы (цена ~11,745). Практически не отличаются на 5m. Или один — предыдущий контракт.
2. **NM** — Норникель (акции). 87.5% WR — лучший одиночный результат.
3. **NR** — товарный фьючерс. PF 9.17 с DD 0.7%
4. **BR** — Brent. PF 4.91, 69 сигналов, DD 1.7%. Идеальный для портфеля.
5. **SBERF** — Сбер-фьючерс. 72.4%, PF 3.44, 29 сигналов.
6. **MM** — Магнит. 72.2%, PF 3.58.

### Не работают
KC, DX, HS (слишком гладкие/мало баров с wide-range), GZ (WR<50%), RI (PF 1.19), Si (53%/1.44)
