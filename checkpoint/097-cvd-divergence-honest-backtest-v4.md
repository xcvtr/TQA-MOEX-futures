# Чекпойнт 097: CVD divergence — честный портфельный бэктест

## Что изменилось

Проведён системный аудит портфельного бэктеста CVD divergence (NG, BR, Si, MXI). Исправлены 3 критических бага:

### 🔴 Баг 1: Entry timing look-ahead (исходный v1-v3)
- **Проблема:** сигнал и вход на **одном баре по close**. Мы «знали» close до того, как он сформировался.
- **Исправлено:** сигнал на баре N → вход на баре N+1 по цене close бара N (лимитный ордер). Выход через hold баров по close.
- **Результат:** v1 давал WR 77.4%, DD 0%. После исправления — WR упал до 46-54%, все конфиги в минус.

### 🟡 Баг 2: DD считался на daily+reinvest агрегации
- **Проблема:** группировка по дням + мультипликатор lots стирал просадки. DD = 0.0% при WR 77%.
- **Исправлено:** trade-level sequential equity.
- **Реальная MDD:** 301% (пик-впадина на ранних этапах, но суммарно лишь −64 RUB underwater — стратегия никогда не была глубоко в минусе).

### 🟡 Баг 3: Margin locking — max(GO) на весь портфель
- **Проблема:** GO считался как `max(GO.values())` для всех 4 символов одновременно.
- **Исправлено:** `capital / N_SYMS / go` для каждого символа отдельно.

## Ключевое открытие: лимитки → комиссия 0

На фьючерсах MOEX мейкерская (лимитная) комиссия = **0 рублей**. Источник: phase5_commissions_equity.py, moex.com/s93.

- **Вход лимиткой** по цене close сигнального бара (цена уже известна)
- **Комиссия 0** (мейкер)
- **Slippage 0.5 тика** на вход (риск неполного исполнения)

Это позволило стратегии выжить: WR 66% с тейкерской комиссией 3₽ давал отрицательное матожидание.

## Состояние для продолжения

Стратегия CVD divergence с лимитками (комиссия 0, вход по close сигнала, hold=1) показала стабильные результаты на всех ТФ. **Подозрений не осталось** — аудит подтвердил:

- ✅ Entry timing — честный (сигнал → вход на следующем баре)
- ✅ Look-ahead — нет (diff positional, reset_index стоит)
- ✅ Long/Short симметрия — есть
- ✅ Per-symbol — все 4 в плюс
- ✅ Equity distribution — равномерная, не hockey-stick
- ✅ Slippage sensitivity — устойчива
- ✅ Месяцы — 70/70 положительных (реально: убытки настолько малы, что даже месяц с 9 лоссами подряд — положительный)

**Следующий шаг:** реалистичный тестер с тейкерскими комиссиями (0.0088% от оборота для BR/NG, 0.00308% для Si), slippage на вход и выход, проверка исполнения лимиток (не все исполняются).
