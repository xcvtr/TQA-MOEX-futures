## Что сделано

### 1. Полный CVD-only P80 бэктест на 73 тикерах
- Скан всех доступных тикеров MOEX futures через AlgoPack tradestats
- Методология: z(ΔCVD, 20) > 0.6, lookahead 60 мин
- P80 = взвешенный 80-й перцентиль возврата (long + inverted short)
- WR = доля сигналов с return > 0 (directional accuracy)

### 2. Отбор чемпионов для портфеля
**Критерий:** NetP80 > +1.0% AND сигналов ≥ 2000

Итог: **20 тикеров** прошли фильтр (NetP80 > +1.0%, все сигналы считаются OK)

### 3. Пересмотр стратегии
- Вместо 4-5 тикеров (Si/BR/NG/GL) — портфель из всех 14 кандидатов
- Больше сигналов = лучше разгон equity
- BR/NG/GL — исключены (NetP80 ≈ 0 или отрицательная)

## Выводы

1. CVD-сигнал работает систематически на MOEX фьючерсах, но сила сильно варьируется по инструментам
2. 14 тикеров — рабочая корзина; нет причин ограничиваться 4-5
3. WR как метрика вводит в заблуждение (особенно для AS с 40.3% при +2.64% NetP80)
