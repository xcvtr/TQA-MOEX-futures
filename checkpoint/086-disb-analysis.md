# Чекпойнт 086 — Анализ дисбаланса агрессивных сделок (disb)

**Дата:** 2026-06-12
**Время:** ~18:30 MSK

## Что сделано

1. **Написан `scripts/analyze_disb.py`** — анализ дисбаланса агрессивных сделок (disb) как предиктора цены на MOEX фьючерсах.
   - Данные: `moex.tradestats_fo` (ClickHouse 10.0.0.63)
   - 1-мин бары, корреляции disb_lag → next_return, oi_diff_lag → next_return
   - Ресемплинг на 5-минутки
   - Простая стратегия long/short по порогу disb (0.3–0.9)
   - Сравнение disb vs oi_diff (комбинированные стратегии с перебором threshold)

2. **OpenCode-делегация** — первая версия написана через OpenCode (бесплатно). Исходные недостатки и их исправление:
   - **Look-ahead bias** — oi_diff без фильтра выбросов давал +2,9M%. Исправлено: rolling 20σ + clip 1–99%
   - **Диагностика** — большинство тикеров не находились. Добавлен `SELECT DISTINCT asset_code` с fuzzy match
   - **Threshold-based сигналы** — комбинированная стратегия считала по всем барам, а не по сигнальным. Переписана с дискретными сигналами
   - **Порядок строк** — `trades` определялся до `strategy_return` → `KeyError`. Пофикшено.
