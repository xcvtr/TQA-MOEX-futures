# 032 — OOS Validation: BASE v2 survives all periods (2026-06-15)

## Контекст
На предыдущем шаге (checkpoint 031) нашли оптимальные параметры BASE через grid search:
- score>0.10, bars=8, stop=1.0ATR, lev=0.50

Перед дальнейшим движением (walk-forward, Kelly) нужно было подтвердить, что результат не переоптимизирован под 2025.

## Методология
4 периода тестирования на всех 7 тикерах портфеля:
1. **2024 OOS** (чистый out-of-sample — оптимизация велась на 2025)
2. **2025 INS** (in-sample — совпадает с периодом оптимизации)
3. **FULL** (2024-01 — 2026-05)
4. **2023_24** (ранний период, до оптимизации)

Система: bar-level simulation, MTM equity, stop loss по ATR, bars_left exit.
Реинвест: капитал реинвестируется через leverage (доля капитала на сделку).
