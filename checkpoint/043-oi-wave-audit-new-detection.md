# 043 — OI Wave Audit: Новый метод детекции

## Статус
- ⚠️ Волновой TROUGH→LONG детектор (по % yur_net) признан неработоспособным
  - SN: находил 36 сделок из 467 экстремумов — пропускал 92%
  - AL, AU — убыточны после walk-forward
- ✅ Новый метод: **PCT_v95_yb90** (volume ≥ 95% + yur_buy ≥ 90% перцентиль, rolling 20)
  - WR=53%, avg return +0.43%, 44k сигналов на 64 тикера
  - Объём (volume) + абсолютный рост yur_buy — главные индикаторы крупных покупок
- TRIZ-выводы:
  1. Перцентильные пороги > z-score
  2. Горизонт 40-80 баров (3-6 часов) оптимален
  3. Проскок (entry на следующем баре) даёт +0.5-1% WR
  4. Stacked confirmation ухудшает (перефильтрация)
  5. YNE (yur_net_pct < -80%) неработоспособен — шум
- Новые кандидаты (WR > 52%): **BM, DX, IB, GD, CE, AF, Eu**

## Что сделано
- Анализ 64 тикеров × 3 типа сигналов × 4 горизонта (13892 строк сводки)
- Перебор 55 TRIZ-конфигураций
- Выявлен лучший метод детекции и набор тикеров
