# Checkpoint 049: TRIZ Diamond v2 — переосмысленный поиск с CBR-фильтром, ATR-выходом, dv_threshold

## Что сделано

- **Анализ текущих результатов**: портфель из PT/SP/CC/AF давал ~34% годовых при DD 1.4% — не 300%
- **TRIZ-переосмысление**: добавлены 3 новых фильтра:
  1. CBR filter — исключение дней ±2 от дат ставки ЦБ РФ (макро-шум)
  2. ATR-adaptive hold — удержание подстраивается под волатильность
  3. dv_threshold — отсев слабых сигналов (только объёмные всплески)
- **Скрининг**: 23 тикера × 5 паттернов × 3-4 параметра × 4 комбинации фильтров = 1,298 комбинаций
- **Найдено 73 «алмаза»** (Calmar ≥ 3, ret ≥ 20%, trades ≥ 10)
- **Портфельная симуляция**: RI + USDRUBF + GL + NM → +168.6% за 2 года (~72% годовых), DD 3.6%, Calmar 47.4

## Результаты

- **Лучший одиночный сигнал**: RI vol_up_oi_up_yb_up hold=5 sl=1% — +173.8%, DD 7.1%, Calmar 24.5, 49 сделок
- **Лучший портфель (100K)**: RI + GL + USDRUBF + NM → +168.6% за 683 дня, DD 3.6%, Calmar 47.4
- **Комплементарность**: RI-USDRUBF всего 5-6% пересечения сигналов — естественная пара хеджирования
- **CBR filter** даёт стабильность: дни ставки ЦБ убирают макро-шум
- **ATR-adaptive hold** — правильная идея, даёт больше сделок в низкой волатильности
- **dv_threshold** — полезен для PD, SV, убирает шумовые сигналы
