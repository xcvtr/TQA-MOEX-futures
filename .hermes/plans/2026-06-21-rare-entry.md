# Rare Entry MOEX Strategy

> **Проблема:** 10-22K сделок за 16 мес — это шум, а не сигнал. Score с весами пропускает 20% баров.
> **Гипотеза:** Киты входят редко. Сигнал должен быть редким (50-200 сделок/год на тикер) и сильным (WR>55%, avg trade > 10× комиссии).

## Подход: жёсткие фильтры вместо score с весами

Вход только когда **ВСЕ** условия выполнены одновременно:

### Базовые фильтры (обязательные)

1. **ADX > 25** — трендовый рынок, не флэт
2. **Объём > 2× среднего** (`vr = volume / vma20 > 2.0`)
3. **OI anomaly** — одно из:
   - `oi_accel_z > 1.5` (z-score ускорения OI ratio)
   - `oi_r_z > 2.0` (z-score OI ratio)
   - `fiz_yur_delta > 0.8` (сильное расхождение физлиц/юрлиц)

### Таймфреймы
- H1 (60min) — основной
- H4 (240min) — для сравнения

### Параметры
- stop_atr: [1.0, 1.5, 2.0, 3.0]
- bars_left: [8, 13, 21, 34]
- lot: [0.10, 0.15, 0.20, 0.25, 0.50]
- slippage: 0.1%

### Направление
- LONG: объёмный всплеск + OI рост юрлиц
- SHORT: объёмный всплеск + OI падение юрлиц

## Критерий успеха
- 50-200 сделок/год на тикер
- CAGR > 0% с комиссиями 4₽ + slippage 0.1%
- WR > 50%
- Calmar > 1.0

## Процедура

1. Взять `scripts/stress_test_full.py` как базу
2. Заменить логику входа: не score с весами, а жёсткие условия
3. Перебрать комбинации фильтров × параметры
4. Если нашлась хотя бы одна работающая комбинация — per-symbol breakdown
5. Если нет — расширить фильтры (уменьшить пороги)
